# 团队 Skills Market 上线公告 🎉

各位团队成员：

我们正式上线了**团队 Skills Market** - 一个共享 AI Agent Skills 的平台，帮助大家提升开发效率！

## 📖 这是什么？

这是一个基于 Plexus 的团队 Skill 分享仓库，包含：
- ✅ **代码审查助手** - 自动审查代码质量
- ✅ **API 测试生成器** - 自动生成测试用例
- ✅ **技术文档生成器** - 根据代码生成文档
- ✅ **Git 提交助手** - 生成规范的 commit message

以及更多由团队成员贡献的实用 Skills！

## 🚀 快速开始（3分钟）

### 1. 安装 Plexus

```bash
npm install -g plexus-agent-config
```

### 2. 订阅团队仓库

```bash
plexus join https://github.com/ybloveavril/team-skills-market.git
```

### 3. 同步配置

```bash
plexus pull
plexus sync
```

### 4. 在 AI 工具中使用

打开 Claude Code、Cursor 或其他支持的 AI 工具，尝试：

```
"请使用 code-reviewer 审查我的代码"
```

或

```
"使用 api-tester 为这个端点生成测试"
```

## 📱 可选：启动 Web Dashboard

```bash
plexus
```

然后访问 http://localhost:7777 浏览和管理 Skills。
